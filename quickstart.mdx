---
title: "Quickstart"
description: "Get 2048 - The Game running on your machine in minutes — as a desktop app or in your browser."
---

<Note>
  Both versions require **Python 3.x**. Check your version with `python --version` or `python3 --version` before proceeding.
</Note>

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/NiwinKumar/2048-TheGame
    cd 2048-TheGame
    ```
  </Step>

  <Step title="Install dependencies">
    Choose the version you want to run and install the required packages.

    <Tabs>
      <Tab title="Desktop App">
        The desktop version uses [NumPy](https://numpy.org) for the game grid and Tkinter, which ships with Python.

        <CodeGroup>
          ```bash pip
          pip install numpy
          ```

          ```bash pip3
          pip3 install numpy
          ```
        </CodeGroup>
      </Tab>
      <Tab title="Web App">
        The web version uses [Flask](https://flask.palletsprojects.com) to serve the game in your browser.

        <CodeGroup>
          ```bash pip
          pip install flask
          ```

          ```bash pip3
          pip3 install flask
          ```
        </CodeGroup>
      </Tab>
    </Tabs>
  </Step>

  <Step title="Run the game">
    <Tabs>
      <Tab title="Desktop App">
        ```bash
        python 2048.py
        ```

        A window titled **2048** opens immediately. Use the arrow keys or **WASD** to move tiles.
      </Tab>
      <Tab title="Web App">
        ```bash
        python app.py
        ```

        Once the server starts, open your browser and go to:

        ```
        http://localhost:5000
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
